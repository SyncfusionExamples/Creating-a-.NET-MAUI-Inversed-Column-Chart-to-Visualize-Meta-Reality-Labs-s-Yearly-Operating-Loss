# Creating an Inversed Column Chart by demonstrating Meta's Reality labs yearly operating loss
The inverted column chart is a commonly used chart type for plotting comparative data values in reverse order. This sample demonstrates how to create an Inverted Column Chart using .NET MAUI Syncfusion charting control - SfCartesianChart to compare Metas Reality Labs' yearly operating losses from 2019 to 2022.
